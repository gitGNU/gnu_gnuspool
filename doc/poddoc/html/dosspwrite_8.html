NAME

dosspwrite - write messages to Windows PC

SYNOPSIS

dosspwrite options

DESCRIPTION

Dosspwrite sends messages to Windows PCs similar to spwrite(8) does for user's terminals in response to the equivalents of the -w and -A options of gspl-pr(1). This is only done for jobs which originated on Windows PCs.

The Windows PC must be running sprserv for this to be effectual.

If the job was submitted by a user working from a client with a DHCP-allocated IP address, a message may be received on all clients currently logged-in with that user name.

FILES

int-config message file

ENVIRONMENT

SPHELPDIR

alternative location for help file directory.

SEE ALSO

spwrite(8), spmdisp(8).
