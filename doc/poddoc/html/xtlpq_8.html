NAME

xtlpq - support displaying job list on LPD protocol servers

SYNOPSIS

xtlpq -p ptr -H host [ -P port ] [ -S host ] [ -l ]

DESCRIPTION

xtlpq obtains and outputs a list of pending jobs on an external print server supporting the LPD protocol. It is invoked by gspl-lpq(1) and is not intended for general use.

OPTIONS

-p ptr

Specify printer.

-H host

Specify host name.

-P port

Specify port name or number, if not standard.

-item -S host

Specify outgoing IP address.

-l

Produce a long listing.

SEE ALSO

gspl-lpq(1), spsyntax(5), xtlpc(8), xtlprm(8).

DIAGNOSTICS

Various diagnostics are read and printed as required on standard error.
