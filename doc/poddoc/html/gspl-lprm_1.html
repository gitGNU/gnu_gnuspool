NAME

gspl-lprm - delete jobs on LPD protocol servers

SYNOPSIS

gspl-lprm printer job ...

DESCRIPTION

gspl-lprm deletes pending jobs on an external print server supporting the LPD protocol.

OPTIONS

Note that the order of treatment, letters and keywords described below may be modified by editing the file splpd.help - see spsyntax(5).

The environment variable on which options are supplied is GSPL_LPRM and the environment variable to specify the help file is LPDINTCONF.

-? or +explain

Causes a summary of the other options to be displayed without taking further action.

+freeze-current

Save all the current options in a .gnuspool file in the current directory.

+freeze-home

Save all the current options in a .gnuspool file in the user's home directory.

FILES

~/.gnuspool configuration file (home directory)

.gnuspool configuration file (current directory)

splpd.help message file

ENVIRONMENT

SPLPRM

space-separated options to override defaults.

LPDINTCONF

location of alternative help file.

SEE ALSO

gspl-lpq(1), spsyntax(5), xtlpc(8).

DIAGNOSTICS

Various diagnostics are read and printed as required from the message file, by default splpd.help.
