NAME

xtlprm - support removing jobs on LPD protocol servers

SYNOPSIS

xtlprm -p ptr -H host [ -P port ] [ -S host ] [ -u user ]

DESCRIPTION

xtlprm deletes pending jobs on an external print server supporting the LPD protocol. It is invoked by gspl-lprm(1) and is not intended for general use.

OPTIONS

-p ptr

Specify printer.

-H host

Specify host name.

-P port

Specify port name or number, if not standard.

-item -S host

Specify outgoing IP address.

-u user

Specify a user name.

SEE ALSO

gspl-lprm(1), spsyntax(5), xtlpc(8), xtlpq(8).

DIAGNOSTICS

Various diagnostics are read and printed as required on standard error.
