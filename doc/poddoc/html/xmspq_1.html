NAME

xmspq - X/Motif spool queue view

SYNOPSIS

xmspq

DESCRIPTION

Xmspq is a fully interactive Motif alternative to the standard print queue manager spq(1). As with spq(1) the format of the screen display, the help messages and the command keystrokes can be easily altered to suit your requirements.

Unlike spq(1) there are no specific command line options to xmspq. The facility to change or specify resources settings for an X11 (and hence Motif) program on the command line can be used.

FILES

~/.gnuspool configuration file (home directory)

.gnuspool configuration file (current directory)

xmspq.help message file

/var/spool/gnuspool spool directory

gnuspoolptrs printers directory

spshed_reps error log file

ENVIRONMENT

XMSPQCONF

path name of alternative message file.
