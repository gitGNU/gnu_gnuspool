NAME

gspl-xmpg - X/Motif spool queue view

SYNOPSIS

gspl-xmpg

DESCRIPTION

Gspl-Xmpg is a fully interactive Motif alternative to the standard print queue manager gspl-pq(1). As with spq(1) the format of the screen display, the help messages and the command keystrokes can be easily altered to suit your requirements.

Unlike gspl-pq(1) there are no specific command line options to gspl-xmpg. The facility to change or specify resources settings for an X11 (and hence Motif) program on the command line can be used.

FILES

~/.gnuspool configuration file (home directory)

.gnuspool configuration file (current directory)

gspl-xmpg.help message file

gnuspoolptrs printers directory

spshed_reps error log file

ENVIRONMENT

GSPL-XMPGCONF

path name of alternative message file.
