NAME

gspl-ptrinstall - install printers interactively

SYNOPSIS

gspl-ptrinstall

DESCRIPTION

Gspl-ptrinstall is a fully interactive script to set up and install printers for GNUspool.

It is written in Perl, but if the user has Python and Python-QT4 installed, will use a GUI display.

SEE ALSO

gspl-padd(1), gspl-getsnmp(1).
